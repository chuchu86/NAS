# NAS — Аналитика охватов VK (апрель–июнь 2026)

Разбор падения охватов группы NewArtSchool (vk.com/newartschool) за апрель–июнь 2026,
сравнение с конкурентами (Smirnov School, Skills Up School, Home Digital School, XYZ School)
и итоговые выводы + план действий.

## Содержимое
- `index.html` — визуальный отчёт (открывается в браузере).
- `Аналитика охватов VK — июнь 2026.html` — тот же файл с исходным названием.

## Как смотреть
Скачай `index.html` и открой в браузере. Либо, если репозиторий сделать public и включить
GitHub Pages, отчёт будет доступен по прямой ссылке.

> Числовые оценки в отчёте иллюстративны (порядок величин), а не выгрузка из статистики сообщества.
